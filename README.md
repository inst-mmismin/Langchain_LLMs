# Langchain 활용 데모 코드 

## 세팅 

- python : 3.9 
- dependency 

```
pip install -r requirements.txt 
```

## 여러 LLM들의 결과를 한번에 보기 

### 온라인 데모 

https://langchainllms.streamlit.app/

### 로컬 실행 

```
streamlit run SLcompLLMs.py
```
